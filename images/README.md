# Project Images

This folder contains all images and photos related to the Smart Dustbin project.

## Upload Your Photos Here

You can add the following types of images:

### 1. Hardware Setup Photos
- Complete project setup
- Circuit board assembly
- Component connections
- Dustbin container with mounted components

### 2. Circuit Diagrams
- Fritzing diagrams
- Schematic diagrams
- Wiring diagrams

### 3. Testing Photos
- Project in action
- Sensor testing
- Servo motor operation
- LED indicators (if added)

### 4. Component Photos
- Individual component images
- Arduino UNO board
- Ultrasonic sensor
- Servo motor

## How to Upload

### Method 1: Via GitHub Web Interface
1. Click on "Add file" > "Upload files"
2. Drag and drop your images
3. Commit the changes

### Method 2: Via Git Command Line
```bash
# Navigate to your project folder
cd Smart-Dustbin

# Copy your images to the images folder
cp /path/to/your/image.jpg images/

# Add and commit
git add images/
git commit -m "Add project photos"
git push origin main
```

## Recommended Image Formats
- JPEG (.jpg, .jpeg) - For photos
- PNG (.png) - For diagrams and screenshots
- GIF (.gif) - For demonstrations

## Image Naming Convention
Use descriptive names for your images:
- `setup_complete.jpg`
- `circuit_diagram.png`
- `arduino_connections.jpg`
- `dustbin_open.jpg`
- `sensor_testing.jpg`

---

**Note:** Keep image sizes reasonable (< 5MB each) for better repository performance.
